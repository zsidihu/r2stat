Secure & Private

R2Stats is a client-side Cloudflare R2 monitoring tool. Your API tokens are stored locally in your browser's localStorage and are never transmitted to our servers. All requests are proxied via Vercel to the official Cloudflare GraphQL API.

About R2Stats Tracker

Managing Cloudflare R2 storage costs requires tracking your daily operations. Our tool helps developers monitor the R2 Free Tier limits (10M Class B and 1M Class A operations per month) to avoid unexpected bills.

Key features: Real-time Cloudflare GraphQL API integration, usage forecasting, and mobile-ready PWA dashboard for R2 object storage.
