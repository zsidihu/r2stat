Secure & Private

R2Stat is a client-side Cloudflare R2 monitoring tool. Your API tokens are stored locally in your browser's localStorage and are never transmitted to our servers. All requests are proxied via Vercel to the official Cloudflare GraphQL API.

About R2Stat Tracker

Managing Cloudflare R2 storage costs requires tracking your daily operations. Our tool helps developers monitor the R2 Free Tier limits (10M Class B and 1M Class A operations per month) to avoid unexpected bills.

Key features: Real-time Cloudflare GraphQL API integration, usage forecasting, and mobile-ready PWA dashboard for R2 object storage.

R2Stat Support & FAQ

How to get Cloudflare API Token & Account ID?

To use R2Stat, you need an API Token with Cloudflare R2 Read permissions. Go to your Cloudflare Dashboard > My Profile > API Tokens > Create Token. Your Account ID is visible on your R2 overview page sidebar.

Understanding R2 Class A vs. Class B Operations

Class A operations are more expensive and include state-changing actions like PutObject or ListBuckets. The R2 Free Tier includes 1 million Class A ops per month. Class B operations are primarily reads (e.g., GetObject). You get 10 million free Class B ops monthly.

Why monitor R2 usage daily?

Cloudflare R2 provides a generous free tier, but exceeding it can lead to unexpected costs. R2Stats helps you track your consumption in real-time using the Cloudflare GraphQL API, providing a 30-day forecast based on your current activity.
